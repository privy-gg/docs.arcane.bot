# Auto Role Setup

::: tip
Setup is done on the [dashboard](/core/dashboard).
:::

![Auto Roles](/images/roles/autoroles-dashboard.png)

## Auto Roles

Auto roles are automatically assigned to members when they join your server. This is useful if you want members to always have one or more roles such as `Member`.

### Limits

| **Tier** | **Auto Roles** |
| - | - |
| Free | 1 |
| Premium | 10 |
